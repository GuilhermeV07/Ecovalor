# EcoValor

Projeto reestruturado do **EcoValor** com foco em front-end, sem dependências da plataforma/IA anterior.

## O que esta versão entrega
- Landing page institucional do EcoValor
- Dashboard de gestão de resíduos
- Marketplace de resíduos industriais
- Painel ESG
- Área financeira
- Insights com IA (simulação visual/local)
- Autenticação local mockada para demonstração

## Como rodar
```bash
npm install
npm run dev
```

## Estrutura
- `client/src/pages`: páginas principais
- `client/src/components`: layout, cards, widgets e componentes UI
- `client/src/_core/hooks/useAuth.ts`: mock de autenticação local

## Observações
Esta versão foi limpa para uso acadêmico/prototipação e não depende mais de backend, OAuth ou serviços da plataforma anterior.
